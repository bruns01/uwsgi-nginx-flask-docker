## Docker build-file repository
This is intended for automated builds of a docker-image for 
scientific web-apps based on
* NGinX
* Flask
* uwsgi
* Numpy
* Scipy
* Pandas
* Matplotlib

The original docker build environment is from
https://github.com/tiangolo/uwsgi-nginx-flask-docker
